基于注解形式

MyBatis提供了@Select,@Update,@Insert,@Delete四种注解，用来实现增删改查。

为接口生成动态代理类的方式有两种 (二者选其一即可)：

  • 在Mapper接口上使用@Mapper注解
  • 在SpringBoot的启动类上使用@MapperScan(basePackages = “接口所在包路径”)注解
